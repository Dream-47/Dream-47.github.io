使用JsDeliver访问github资源

前言

本想通过将一些通用的js库上传到github，然后通过src访问对应资源，

测试时发现html在运行时会出现CROS跨域问题，

请不要把github当成cdn，

jsDeliver可以解决这个问题

部署

  1. github创建release包

  2. 访问github资源

  3. 如果发布版本有问题，需要重新发布一个版本，同时更新version，cdn会缓存错误资源，这倒比较麻烦
